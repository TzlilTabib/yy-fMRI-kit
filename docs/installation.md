# Installation

## Stable release

To install YY fMRI kit, run this command in your terminal:

```sh
uv add yy_fmri_kit
```

Or if you prefer to use `pip`:

```sh
pip install yy_fmri_kit
```

## From source

The source files for YY fMRI kit can be downloaded from the [Github repo](https://github.com/TzlilTabib/yy_fmri_kit).

You can either clone the public repository:

```sh
git clone git://github.com/TzlilTabib/yy_fmri_kit
```

Or download the [tarball](https://github.com/TzlilTabib/yy_fmri_kit/tarball/master):

```sh
curl -OJL https://github.com/TzlilTabib/yy_fmri_kit/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd yy_fmri_kit
uv pip install .
```
